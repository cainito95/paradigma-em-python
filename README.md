E - É uma linguagem fortemente tipada ou francamente tipada? (qual a diferença? exemplos)

A tipagem (forte vs. fraca) refere-se ao rigor com que uma linguagem trata os tipos de dados.
Por exemplo, o Python é fortemente tipada por não deixa somar número com texto automaticamente, precisaria converter os dados.
Uma linguagem fracamente tipada permite conversões automáticas entre tipos de dados, como o JavaScript.
